DEFINICIONES GENERALES

Etiquetas <meta> mínimas en el <head>

En el <head>, además del charset, el viewport, y el title, es importante que también vaya el meta "description" de no más de 165 caracteres ya que esta descripción es la que aparece como texto debajo de un link de una página en los resultados de búsqueda de google, además si se tiene secciones dentro de un sitio web como "home", "contacto", "acerca de", etc., cada una debe tener un título y una descripción diferente para que goolge muestre estas secciones también en los resultados de búsqueda.

Ejemplo:

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<link rel="stylesheet" href="./styles.css">


<!-- <title> debe tener entre 55 y 65 caracteres -->

<title>DOCUMENTACIÓN HTML</title>


Esta descripción aparece como texto de cada página dentro de los resultados de búsqueda de google. No debe tener más de 165 caracteres

<meta name="description" content="Página dedicada al contenido de ... ">

Data Attributes

Son atributos personalizados que se incluyen en las etiquetas html para su posterior utilización con JavaScript o CSS. Estos atributos deben colarse como cualquier atributo estandar, es decir, dentro de la etiqueta de apertura y empezar con el texto "data-" seguido del identificador del atributo y su respectivo valor.

Ejemplo: en el siguiente párrafo se ha incluido un data attribute llamado "data-parrafo" y su valor es "parrafo-especial"

<p data-parrafo="parrafo-especial">este es un párrafo que contiene un data attribute. Visualmente no se muestra nada diferente a un párrafo común pero en el código se podrá usar posteriormente el data attribute para temas de CSS o JavaScript.</p>

Guía de optimización en buscadores (SEO) para principiantes

Son etiquetas y/o atributos en tu html que permiten al navegador entender mejor el contenido de la página. El efecto principal es que esta página pueda aparecer en los primeros resultados de una búsqueda. Algunas de estas etiquetas y atributos son:

  • rel="canonical" atributo de la etiqueta <link> que sirve para indicar una url canónica.
  • rel="icon" atributo de la etiqueta <link> que sirve para indicar el ícono da la página web.
  • rel="apple-touch-icon" atributo de la etiqueta <link> que sirve para indicar el ícono da la página web en dispositivos móviles.
  • property="og:title" atributo de la etiqueta <meta> para compartir páginas web en redes sociales como un objeto enriquecido (tipo tarjetas por ejemplo). Más info aquí, The Open Graph protocol

Ejemplo:

La etiqueta <hr> normalmente sólo se utiliza para colocar una línea divisoria cosmética en la página, pero en realidad es más que eso, ya que identifica un salto semántico de contenido
