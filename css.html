VARIABLES / CUSTOM PROPERTIES

Se refiere a las típicas variables en los lenguajes de programación, y que también se pueden usar en CSS.

Al igual que en la mayoría de lenguajes de programación, en CSS, es recomendable que las variables sean declaradas al inicio y en la raíz (root) del documento para que tengan el mayor ámbito posible.

Para declarar una variable se debe anteponer dos guiones seguidos antes del nombre, luego se asigna un valor con dos puntos. Para usar el valor de una variable hay que llamar a la función var() y dentro de los paréntesis poner el nombre de la variable.

Referencias:

The Ultimate Guide to CSS Variables Using CSS custom properties like this is a waste

Ejemplo 1: declaración y uso simple de una variable

/* declaracion de una variable en el root llamada '--bg-code' */

:root {

--bg-code: #fff;

}


/* uso posterior de la variable */

.clase {

background-color: var(--bg-code);

}

Es necesario siempre realizar la declaración de una variable. Si por alguna razón no se ha hecho, por ejemplo es una variable que viene de JS, se puede definir un valor 'fallback' para que no haya problema con esa variable, luego ese valor se usará en todo el ambito de la misma. Para declarar un fallback a una variable, hay que colocar una coma luego del nombre de la variable seguido del valor requerido.

Ejemplo 2: valor fallback para una variable

/* no existe declaración de la variable '--full-width' */


/* definición de un fallback para la variable '--full-width' */

.clase {

max-width: var(--full-width, 1100px);

}

También se puede usar dentro de un fallback, un valor de otra variable u otro fallback.

Ejemplo 3: valor de otra variable como fallback, más un dato extra como otro fallback. En este caso la segunda variable no contenga valor alguno por x razón, se usará el último dato.

/* declaracion de una variable en el root llamada '--segunda-variable' */

:root {

--segunda-variable: green;

}


/* si --primera-variable no ha sido declarada, usar --segunda-variable que sí ha sido declarada (su valor es 'green').
Si por alguna razón --segunda-variable no tiene valor usar 'red' */

.clase {

color: var(--primera-variable, var(--segunda-variable, red);

}

El uso de variables con seudo elementos como :hover es muy útil.

Ejemplo 4: cuando se pasa el mouse sobre el elemento, se cambian el valor de las variables. Tener en cuenta que con este método cambiarán todas las referencias a estas variables, si se tienen otros elementos en otras posiciones de la página también cambiarán.

/* declaracion de tres variables */

:root {

--bg-color: #ccc;

--front-color: #000;

--borde: 1px;

}


/* uso de variables */

.clase {

background-color: var(--bg-color);

color: var(--front-color);

border: var(--borde) var(--front-color) solid;

}


/* uso de las variables sobre el seudo elemento :hover */

.clase:hover {

--bg-color: red;

--color: blue;

--border: 3px;

}

SELECTORES AVANZADOS: SEUDO ELEMENTOS

Son elementos abstractos que nos permiten seleccionar 'porciones' del documento que no están específicamente y directamente definidas en el arbol del documento, es decir, no tienen una etiqueta que los defina como tales. Por ejemplo, no son seudo elementos lo que sí están directamente definidos en documento como pueden ser un párrafo que está definido por la etiqueta <p></p>, una imagen por la etiqueta <img>, etc. Mientras que seudo elementos serían los que no tienen una etiqueta específica y por tanto no podemos definirlos exactamente, como serían, la primera línea o primer letra un párrafo, un texto seleccionado por el usuario, un contenido agregado o eliminado bajo ciertas circunstancias, etc.

Los seudo elementos son siempre parte de un 'elemento origen' también llamado 'selector' el cual normalmente sí es una etiqueta, clase o seudo clase.

Para la sintaxis de los seudo elementos primero se debe escribir el selector (elemento origen) respectivo seguido de dobles dos puntos (::) y por último el seudo elemento:

selector::seudo-elemento {

propiedad: valor;

}

También se puede seleccionar un seudo elemento basándose en el estado del elemento origen (selector), por ejemplo p:hover::first-line selecciona el seudo elemento 'primera línea' de un párrafo (p) cuando se posiciona el cursor sobre él (:hover), o también a:hover::after que agregaría un contenido después del link cuando el cursor pase sobre este link, podría ser por ejemplo una imagen de flecha para indicar con mayor énfasis la disponibilidad del link.

SELECTORES AVANZADOS: SEUDO CLASES

Son selectores que se agregan a los elementos para identificarlos de acuerdo a un estado específico, por ejemplo si el cursor está sobre un elemento, o si ya se ha visitado un link, o si un elemento es el primero o último en el árbol de elementos, etc.

Al igual que los seudo elementos, se necesita primero un 'elemento origen' también llamado 'selector' el cual normalmente es una etiqueta o clase.

Para la sintaxis de las seudo clases primero se debe escribir el selector (elemento origen) respectivo seguido de dos puntos (:) y por último la seudo clase:

selector:seudo-clase {

propiedad: valor;

}

:nth-child() :
:nth-last-child() :

Seleccionan elementos basándose en una expresión algebráica (p.e. "n" o "2n" o "4n-1"). :nth-last-child() lo hace en reversa, empezando desde el último elemento. El valor indicado entre los paréntesis indica lo que se quiere seleccionar: elementos únicos (n), pares o impares (even) (odd) o cada tres o los primeros cinco, etc. Las expresiones algebráicas funcionan como fórmulas comunes donde hay que reemplazar 'n' por una numearación consecutiva pudiendo crear selectores más complejos que encuentren elementos a intervalos espaciados con regularidad, mediante la microsintaxis An + B. Por ejemplo:

3n+3 -> reemplazando n por una numeración consecutiva tendríamos una selección de tres en tres:
(3 x 0) + 3 = 3 = 3er elemento
(3 x 1) + 3 = 6 = 6to elemento
(3 x 2) + 3 = 9 = 9no elemento
(3 x 3) + 3 = 12 = 12vo elemento

4n-1 -> reemplazando n por una numeración consecutiva tendríamos una selección de cuatro en cuatro empezando del 3er elemento:
(4 x 0) - 1 = -1 = no hay selección
(4 x 1) - 1 = 3 = 3ro elemento
(4 x 2) - 1 = 7 = 7mo elemento
(4 x 3) - 1 = 11 = 11vo elemento

-n+4 -> reemplazando n por una numeración consecutiva tendríamos una selección de los cuatro primeros elementos:
-0 + 4 = 4 = 4to elemento
-1 + 4 = 3 = 3er elemento
-2 + 4 = 2 = 2do elemento
-3 + 4 = 1 = 1er elemento
-4 + 4 = 0 = no hay seleccion, fin

:nth-of-type() :
:nth-last-of-type() :

Similar a :nth-child, pero actúa sobre elementos que sean del tipo que se especifique. Por ejemplo, dentro de un div tenemos varios párrafos e imágenes combinados y se desea selccionar todas la imágenes impares se usaría img:nth-of-type(odd). Es particularmente util cuando se trabaja con listas o tablas y se está alternando entre elementos <ol>, <ul>, <li>, <dt>, <dd>, etc.

:first-of-type :
:last-of-type :

Selecciona el primer elemento del tipo especificado. Por ejemplo, si tuvieramos dos div y cada uno con varios párrafos, imágenes, listas, y quisieramos seleccionar la primera imagen que aparezca dentro de cada uno de los dos div que tenemos, usaríamos div img:first-of-type

:only-of-type :

Selecciona a un elemento que sea el único hijo de su tipo dentro del padre, es decir, si hay otros hijos del mismo tipo (hermanos del mismo tipo), no seleccionará nada.

:only-child :

Selecciona a un elemento que sea único hijo del padre, es decir, si hay otros hermanos sean o no del mismo tipo, no seleccionará nada.

:has() :

Permite aplicar estilo a un elemento 'padre' (selector) si un hijo cumple una condición. Es como tener un IF al seleccionar un elemento, ya que solo agregará un cierto estilo si cumple con una determinada condición.

selector:has(lista-de-condiciones) {

propiedad: valor;

}

SELECTORES AVANZADOS: SELECTORES POR ATRIBUTOS

[atributo]

Sirve para seleccionar un elemento basado en uno o varios atributos con las características que se describan. El atributo o los atributos escogidos deben ir indicados entre corchetes [atributo: valor] después de elemento o clase.

Ejemplo 1: seleccionar únicamente los elementos 'input' que contengan el atributo 'type' igual a 'email' y definir el color de borde a green.

input [type='email'] {

border-color: green;

}

Ejemplo 2: seleccionar únicamente los elementos 'input' que contengan el atributo 'name' igual a 'nombre' y definir el color de borde a blue.

input [name='nombre'] {

border-color: blue;

}

Ejemplo 3: seleccionar únicamente los elementos 'input' que contengan el atributo 'type' igual a 'text' y además el atributo 'id' igual a 'nombre' y definirles el color de fondo a lightblue.

input [type='text'][id='nombre'] {

background-color: lightblue;

}

Ejemplo 4: seleccionar únicamente los elementos 'input' que contengan definido el atributo 'required' y definirles el color de borde a red.

input [required] {

border-color: red;

}

Ejemplo 5: seleccionar únicamente los elementos 'input' que contengan definido el atributo 'type' igual a 'password' y además el atributo 'maxlength' igual a '8' y definirles el color de fondo a green.

input [type='password'][maxlegth='8'] {

background-color: green;

}

Ejemplo 6: seleccionar únicamente los elementos 'button' que contengan el atributo 'type' igual a 'submit' y definirles el color de fondo a purple.

button [type='submit'] {

background-color: purple;

}

Ejemplo 7: seleccionar únicamente los elementos link (<a>) en los que el atributo 'href' empiece con '#' y definirles el color de fondo a yellow.

a [href^='#'] { /* el símbolo ' ^ ' quiere decir 'empieza con...' */

background-color: yellow;

}

Ejemplo 8: seleccionar únicamente los elementos link (<a>) en los que el atributo 'href' termine con '.com' y definirles el color de fondo a red.

a [href$='.com'] { /* el símbolo ' $ ' quiere decir 'termina con...' */

background-color: red;

}

Ejemplo 9: seleccionar únicamente los elementos link (<a>) en los que el atributo 'href' contenga 'enlace' y subrayarles el contenido.

a [href*='enlace'] { /* el símbolo ' * ' quiere decir 'contiene...' */

text-decoration: underline;

}

SELECTORES AVANZADOS: COMBINADORES

(espacio) Selector Descendente

> Selector hijo

+ Selector hermano adyacente

~ Selector hermano general

Combinadores son reglas que distinguen las diferentes relaciones entre los selectores.

Referencias:

CSS Combinators CSS Combinators CSS Combinators CSS selectors and combinators

padre > selector-hijo :

selecciona sólo a los elementos que coincidiendo con lo indicado en selector-hijo, sean hijos directos del padre, es decir, que no estén anidados dentro de otros elementos hijos. No es necesario que éstos elementos sean adyacentes ni los primeros descendientes del padre, pueden estar ubicados en cualquier lugar dentro del padre pero siempre como hijos directos.

Ejemplo 1: seleccionar a todos los párrafos hijos directos de la clase 'contenedor' y formatearlos en itálica y color gris.

.contenedor > p {

font-style: italic;

color: grey;

}

Ejemplo 2: seleccionar únicamente los elementos de listas desordenadas (ul) hijos directos de la clase 'contenedor' y eliminarles el símbolo de lista.

.contenedor > ul {

list-style: none;

}

Ejemplo 3: seleccionar únicamente los links (<a>) hijos directos de la clase 'contenedor' y definirlos en color red.

.contenedor > a {

color: red;

}

elemento-anterior + hermano-adyacente :

selecciona a un único elemento (sólo a uno) que coincidiendo con lo indicado en hermano-adyacente, sea el elemento exactamente siguiente al indicado en elemento anterior, es decir, tiene que ser el siguiente elemento y sólo uno.

Ejemplo 4: seleccionar dentro de la clase 'contenedor', al único hermano adyacente enlace (a) de un título 1 (h1) y definirle el color red y en negrita.

.contenedor h1 + a {

color: red;

font-weight: bold;

}

elemento-anterior ~ hermano-general :

selecciona a todos los hermanos, es decir, que están en el mismo nivel de jerarquía en el DOM, del tipo hermano-general que aparezcan despúes del indicado en elemento-anterior. Estos hermanos pueden o no ser los exactamente siguientes o consecutivos.

Ejemplo 5: seleccionar dentro de la clase 'contenedor', a todos los hermanos que aparezcan después de un título 1 (h1) y definirles el color purple.

.contenedor h1 ~ a {

color: purple;

}
